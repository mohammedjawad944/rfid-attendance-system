# RFID Attendance System

This project implements a contactless attendance system using RFID technology and Arduino. It automatically records attendance when an RFID card is scanned.

## Components
Arduino Uno
MFRC522 RFID Module
RFID Cards
OLED Display
Jumper Wires

## Features
Contactless attendance recording
Fast RFID card scanning
Automatic attendance tracking
Can be connected to cloud or database

## Technologies Used
Arduino Programming
RFID Technology
Embedded Systems

## Applications
School attendance systems
College attendance systems
Office entry systems
